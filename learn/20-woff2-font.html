#20 WOFF2 + font-display

影響指標: FCP, LCP | 難易度: 基礎

1. フォント形式の比較

形式圧縮サイズ目安ブラウザ対応
TTF/OTFなし6.3MB全ブラウザ
WOFFzlib (gzip相当)~3MB全ブラウザ
WOFF2Brotli~2MB (さらにサブセットで24KB)全モダンブラウザ

WSH 2026 では OTF 13MB (2ファイル) → WOFF2 46KB に削減。99.65%のサイズ削減。

2. サブセット化

フルフォントには数千〜数万のグリフ（文字の形状データ）が含まれます。実際に使う文字だけを抽出することをサブセット化と呼びます。

# pyftsubset (fonttools) を使ったサブセット + WOFF2 変換
pip install fonttools brotli

# 使用する文字を指定してサブセット化
pyftsubset input.otf \
  --text="利用規約第一条本サービスについて..." \
  --flavor=woff2 \
  --output-file=output.woff2

# または Unicode 範囲で指定
pyftsubset input.otf \
  --unicodes="U+3000-30FF,U+4E00-9FFF" \
  --flavor=woff2 \
  --output-file=output.woff2
サブセット化の注意点

サブセット化すると、含まれない文字はフォールバックフォントで表示されます。WSH 2026 では利用規約ページの見出し96文字だけに絞りましたが、動的に変わるテキストに使うフォントのサブセット化は慎重に行う必要があります。

3. font-display の4つの値

font-display のタイムライン: block period swap period failure ──────|───────────────|────────────────|───────────▶ └ フォント未読込 ┘ block: ████████████████████████████████████████████ テキスト非表示 │ カスタムフォント表示 (最大3秒) │ swap: █ │ フォールバック表示 → カスタムフォント (極短い block) │ (FOUT: Flash of Unstyled Text) fallback: ██████ │ フォールバック表示 → 条件付き切替 (短い block) │ (swap period が短い) optional: ██████ │ フォールバック表示（切替なし） (短い block) │ (次回アクセスで使用)
値動作使いどころ
blockフォント読込まで最大3秒テキストを非表示 (FOIT)アイコンフォント（文字化けを避けたい）
swap即座にフォールバック表示、後からカスタムフォントに切替 (FOUT)本文テキスト、見出し
fallback100ms非表示、その後フォールバック。短い swap 期間重要だが FOUT を最小限にしたいテキスト
optional100ms以内に読み込めなければフォールバックのまま。次回以降で使用装飾的なフォント

4. WSH 2026 での変更

Before
@font-face {
  font-family: "Rei no Are Mincho";
  font-display: block;  /* ← 最大3秒テキスト非表示！ */
  src: url(/fonts/ReiNoAreMincho-Regular.otf) format("opentype");
  /* 6.3MB の OTF をダウンロード */
}
After
@font-face {
  font-family: "Rei no Are Mincho";
  font-display: swap;   /* ← 即座にフォールバック表示 */
  src: url(/fonts/ReiNoAreMincho-Regular.woff2) format("woff2");
  /* 24KB の WOFF2 をダウンロード */
}

5. FOIT vs FOUT

  • FOIT (Flash of Invisible Text): font-display: block のデフォルト動作。フォント読込中はテキストが見えない。ユーザーは空白のページだと思う
  • FOUT (Flash of Unstyled Text): font-display: swap の動作。フォールバックフォントで先に表示し、カスタムフォントが来たら切り替える。一瞬フォントが変わるが、テキストは常に読める

パフォーマンス指標（FCP/LCP）の観点では、テキストが早く表示される swap が圧倒的に有利です。

確認クイズ: WOFF2 を gzip 圧縮する意味はある？

ありません。WOFF2 は内部で Brotli 圧縮されているため、さらに gzip しても数バイト増えるか同程度です。Express の compression ミドルウェアはデフォルトで WOFF2 の Content-Type を圧縮対象外にしています。

確認クイズ: font-display: optional はいつ使う？

装飾的なフォントで、FOUT（フォント切替時のちらつき）が気になる場合に使います。初回訪問ではフォールバックのまま表示し、バックグラウンドでフォントをキャッシュ。2回目以降の訪問でカスタムフォントが使われます。CLS（Cumulative Layout Shift）が0になるメリットがあります。
